# Sales Overview Dashboard | Power BI

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de transformar dados de vendas em informações estratégicas para apoiar a tomada de decisão.

Durante o desenvolvimento, busquei ir além da criação de gráficos e focar em compreender o contexto do negócio. Antes de construir qualquer visualização, procurei responder uma pergunta fundamental: **quais informações um gestor realmente precisa para tomar decisões?**

A partir dessa abordagem, o dashboard foi estruturado para apresentar indicadores claros e interativos, permitindo analisar o desempenho comercial por diferentes perspectivas.

---

## Contexto de Negócio

Imagine uma empresa que atua em diferentes regiões, comercializando diversas linhas de produtos e atendendo clientes distribuídos por várias localidades.

A gestão precisa acompanhar constantemente indicadores como receita, custos, margem de lucro e desempenho das unidades para identificar oportunidades de crescimento e melhorar a eficiência operacional.

Nesse cenário, um dashboard executivo permite transformar grandes volumes de dados em informações úteis para decisões mais rápidas e assertivas.

---

## Perguntas de negócio respondidas

O dashboard foi desenvolvido para responder questões como:

- Qual é a receita total da empresa?
- Quanto foi gasto para gerar essa receita?
- Qual é a margem de lucro?
- Quais linhas de negócio apresentam melhor desempenho?
- Como as vendas evoluem ao longo do tempo?
- Quais plantas concentram maior faturamento?
- Onde estão localizados os clientes?

## Principais aprendizados

Este projeto reforçou que a análise de dados vai além da construção de dashboards.

Ao longo do desenvolvimento, compreendi a importância de:

- entender o problema de negócio antes de iniciar a análise;
- transformar dados em informações relevantes para a tomada de decisão;
- construir indicadores que realmente agreguem valor aos gestores;
- utilizar Power Query para preparar dados de forma eficiente;
- criar medidas em DAX para gerar métricas de desempenho;
- desenvolver dashboards intuitivos, priorizando a experiência do usuário.

Mais do que aprender novas funcionalidades do Power BI, este projeto fortaleceu minha visão de que um bom analista precisa conectar dados, contexto e objetivos do negócio para gerar insights que apoiem decisões estratégicas.
